ArcRotateCamera

new ArcRotateCamera(name, alpha, beta, radius, target, scene)

Create a new ArcRotateCamera. ArcRotateCamera is a camera that rotates around a given pivot. It can be controlled with the mouse or touch events (and it also requires hand.js to work) A tutorial about ArcRotateCamera can be found here

Parameters

name string
alpha number
beta number
radius number
target any
scene Scene
